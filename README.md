## 📝〉Présentation

XYEngine est un moteur de jeu 2D, développé sans grande prétention.

L'objectif est d'avoir un moteur fonctionnel pour mes divers projets de jeux vidéo. Mais aussi pour apprendre et comprendre comment fonctionne un moteur de jeu.
